goodsDomainPath String 商品主图

goodsMainphotoPath String 商品主图片

goodsPayoffPrice String goodsPayoffPrice
goodsCommissionPrice String goodsCommissionPrice
goodsCommissionRate String goodsCommissionRate
storeDomainPath String storeDomainPath
goodsCount Integer 商品数量
goodsReturnStatus String 退货商品状态|-2为超过退货时间未能输入退货物流 -1为申请被拒绝，1为可以退货 5为退货申请中 6为审核通过可进行退货 7为退货中，10为退货完成,等待退款，11为平台退款完成
goodsMobilePrice BigDecimal 手机专享价
goodsIntegralPrice BigDecimal 积分价
