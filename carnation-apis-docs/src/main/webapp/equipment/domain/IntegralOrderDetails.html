IntegralOrderDetails

简要描述

订单详情

属性

名称 类型 描述
id Long 订单id
igoOrderSn String 订单号
orderStatus Integer 订单状态|0为已提交未付款，20为付款成功，30为已发货，40为已收货完成,-1为已经取消，此时不归还用户积分
status String 订单状态
igoTotalIntegral Integer 总共消费积分
igoTransFee BigDecimal 购物车运费
addTime Date 时间
igoPayment String 支付方式，使用mark标识
payment String 支付方式
igoPayTime Date 支付时间
igoMsg String 兑换附言
receiverName String 收货人姓名,确认订单后，将买家的收货地址所有信息添加到订单中，该订单与买家收货地址没有任何关联
receiverArea String 收货人地区,例如：辽宁省沈阳市铁西区
receiverZip String 收货人邮政编码
receiverAreaInfo String 收货人详细地址，例如：凌空二街56-1号，4单元2楼1号
receiverTelephone String 收货人联系电话
receiverMobile String 收货人手机号码
igoExpressInfo String 物流公司信息
expressInfo String 物流公司信息
igoShipCode String 物流单号
igoShipTime Date 发货时间
igoShipContent String 发货说明
igoList IGo[] 礼品信息
