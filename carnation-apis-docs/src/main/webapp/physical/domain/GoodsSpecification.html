API文档 体检宝 GoodsSpecification

GoodsSpecification

简要描述

商品规格

属性

名称 类型 描述
id Long id
name String
specType Integer 规格类型，0为平台自营规格，1为商家规格，商家规格时需要有对应的店铺
type String 规格类型(文字或图片)
goodsClassId Long 规格对应的主营商品分类,该分类为平台二级商品分类（level=1）
storeId Long 当规格类型为商家规格时对应的店铺
specProperties GoodsSpecProperty[] 商品规格属性1
properties GoodsSpecProperty[] 商品规格属性
