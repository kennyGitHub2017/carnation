API文档 商户 ReturnGoodsLog

ReturnGoodsLog

简要描述

退货日志

属性

名称 类型 描述
id Long id
addTime Date 申请时间
expressCode String 快递号
expressCompanyName String 快递公司名称
expressCompanyId Long 快递公司 id
goodsAllPrice String 商品总价格
goodsId Long 退货的商品id
goodsGspIds String 退货的商品 sku ids
goodsMainphotoPath String 商品图片路径
goodsName String 商品的名称
goodsPrice String 商品价格
goodsReturnStatus String 退货商品状态 -2为超过退货时间未能输入退货物流 -1为申请被拒绝，1为可以退货 5为退货申请中 6为审核通过可进行退货 7为退货中，10为退货完成,等待退款，11为平台退款完成
goodsType Integer 商品的类型 0为自营 1为第三方经销商
refundStatus Integer 退款状态 0为未退款 1为退款完成
returnContent String 退货描述
returnOrderId Long 商品对应的订单id
returnServiceId String 退货服务单号
selfAddress String 收货时向买家发送的收货地址，买家通过此将货物发送给卖家
storeId Long 商品对应的店铺id
